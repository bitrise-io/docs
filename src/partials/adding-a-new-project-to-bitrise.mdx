1. On the **Dashboard**, click **New project** and select **Configure Bitrise CI**.
1. In the **Repository** stage: select your Git provider and connect your repository, set up SSH key access if needed, then select a default branch.
1. In the **CI configuration** stage: Bitrise scans your repository and generates a starter configuration. If your project type is detected, it appears under **Detected options**. If not, select it manually from the **Project type** dropdown.
1. In the **Build settings** stage: review the pre-selected build machine and <GlossTerm baseform="stack">stack</GlossTerm>, set access to builds (private or public), and optionally register a webhook.
1. Click **Finish**.

For the full step-by-step guide, see [Adding a new project](/en/bitrise-ci/getting-started/adding-a-new-project).
