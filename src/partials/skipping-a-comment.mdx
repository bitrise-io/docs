## Skipping a comment

PR comments can trigger builds but if you want to prevent that, you can skip a build by adding `[skip ci]` / `[ci skip]` or `[skip bitrise]` /`[bitrise skip]` in the build's PR comments.

If you use tools that automatically comment on PRs and that starts a build, configure your tool to include a skip command in its comments.
