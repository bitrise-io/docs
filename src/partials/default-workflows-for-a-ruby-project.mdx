## Default Workflows for a Ruby project

| Workflow ID | Workflow summary | Workflow description |
| --- | --- | --- |
| `run_tests` | Run your project’s tests. | The Workflow clones your Git repository and restores your gem cache with the `gem-{{ checksum "Gemfile.lock" }}` caching key. It installs dependencies depending on signals in your codebase:  - If MySQL gem is detected, it installs system dependencies. - If Bundler is detected, it installs dependencies with `bundle install`.  If a relation database gem is detected, it performs database setup: `db:create db:schema:load`.  If a test framework or a Rakefile is detected, the Workflow runs your tests.  The Workflow always saves the Ruby gem cache and includes the **Deploy to Bitrise.io** Step. |
