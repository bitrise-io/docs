## Configuring priority when starting a build manually

You can set a priority when [starting a build manually](/en/bitrise-ci/run-and-analyze-builds/starting-builds/starting-builds-manually.html). The priority only takes effect if other builds are queued when you start a build.

The priority set when manually starting the build overrides all other priorities.

1. Open Bitrise CI and select your project.
1. On the main project page, click **Start build**.
1. Select the **Advanced** option.

   ![manual-build-priority.png](/img/_paligo/uuid-9fdfc68c-9744-f7ae-fee6-b4cc984bdf3e.png)
1. Find the **Priority** field and set an integer value between -100 and 100. The default value is 0.
