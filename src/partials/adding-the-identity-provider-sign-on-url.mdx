## Adding the identity provider sign-on URL

1. Log in to Bitrise and hover over the left navigation bar.
1. Make sure you have the right workspace selected in the **Workspace** menu.
1. Select **Settings**.

   ![workspace-settings.png](/img/_paligo/uuid-b9660c7d-79af-481d-c05a-17356619dd07.png)
1. Go to the **Single Sign-on** tab.
1. Add the **Identity provider sign-on URL** from AD FS in the **SAML SSO provider Single Sign-On URL (SSO URL)** field.

   For example, a valid value is `https://<AD FS URL>.com/adfs/ls`.
