## Additional linked repositories for GitHub Enterprise Server

You can link additional repositories to a project that uses GitHub Enterprise Server. Linking a repository means giving the project access to multiple repositories in addition to the [default repository](/en/bitrise-platform/projects/configuring-the-repository-url-and-the-default-branch).

You can read about configuration details here: [Additional linked repositories](/en/bitrise-platform/repository-access/github-app-integration#additional-linked-repositories-via-a-github-app).
