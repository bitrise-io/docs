:::important[Credit-based plans only]

This article is only relevant if you are on a [credit based (pay-by-minute) subscription plan](http://www.bitrise.io/pricing). Users who aren’t on a credit based plan don’t see the Credits page on Insights.

:::

Keeping an eye on your credit usage can help to avoid surprises caused by changes which result in increased credit usage. Using Bitrise Insights you can see your credit usage trends and if you notice any negative trend you can use Insights to track down what is causing that.
