[Connecting to an Apple service](/en/bitrise-platform/integrations/apple-services-connection/about-connecting-to-apple-services) (such as the App Store Connect or the Apple Developer Portal) with the API key requires generating an API key, adding the authentication data on Bitrise, and assigning the key to the app.

The API key authentication is the recommended way when connecting Bitrise to Apple Services. You can have 50 API keys added to the **Apple Services connection** page but your app can use only one.
