An incoming webhook on Bitrise serves one purpose: to start builds automatically when a certain code event (code push, Git Tag, pull request) happens. You need to register an incoming webhook to your repository and [configure build <GlossTerm baseform="Trigger">triggers</GlossTerm>](/en/bitrise-ci/run-and-analyze-builds/build-triggers/configuring-build-triggers.html).

:::note[Webhook issues]

If your builds triggers don't work, there might be a problem with your webhooks: [Builds aren't getting triggered](https://support.bitrise.io/hc/en-us/articles/360014238237).

:::
