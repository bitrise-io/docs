You can sign up for Bitrise either via an email address or by authenticating yourself with your GitHub, Bitbucket or GitLab account. Signing up using a Git provider account brings some practical benefits, like logging in with one click and of course you won’t have to authorize your Git account when adding repositories hosted by these providers again.
