Every <NT>Amazon EC2</NT> Mac instance needs to run on a dedicated host. To be able to run the <NT>Bitrise</NT> AMI on an <NT>EC2</NT> Mac instance, you'll need to allocate a dedicated host and launch the <NT>EC2</NT> Mac instance on that host. Doing that requires knowing the host ID which <NT>AWS</NT> automatically generates when you allocate a host.

:::note[<NT>AWS</NT> pricing for Mac dedicated hosts]

For macOS instances, a dedicated host must be allocated. On-Demand <NT>EC2</NT> Mac Dedicated Hosts have a minimum host allocation and billing duration of 24 hours. For more details, check out the official <NT>AWS</NT> documentation: [Amazon EC2 Dedicated Hosts Pricing](https://aws.amazon.com/ec2/dedicated-hosts/pricing/#Pricing_for_Dedicated_Hosts).

:::

:::important[Service quotas]

Make sure you have enough service quotas in your selected region to be able to launch a dedicated host.

- [Viewing current quotas](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/ec2-resource-limits.html#view-limits).
- [Request an increase](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/ec2-resource-limits.html#request-increase).

:::

1. On the <NT>EC2</NT> Dashboard, find the **<NT>Instance</NT>s** menu on the left navigation bar.
1. Select **Dedicated <NT>Host</NT>s**.
1. Click **Allocate dedicated host**.
1. Fill out the required fields:

   - **<NT>Name</NT> tag**: This will be the name of your host.

     :::note[Host ID]

     The name is not the same as the host ID which you need to launch an <NT>EC2</NT> instance on the dedicated host. The host ID is automatically generated by <NT>AWS</NT>.

     :::
   - **<NT>Instance</NT> family**: **mac2**
   - **<NT>Instance type</NT>**: **mac2.metal, mac2-m2.metal, mac2-m2pro.metal**
   - **<NT>Availability Zone</NT>**: It depends on the subnets you plan to use. <NT>AWS</NT> will tell you if there is no node capacity in your chosen zone.
   - **<NT>Quantity</NT>**: The number of hosts you want to create.
   - **<NT>Host</NT> maintenance**: Disabled. The `mac2` instance family does not support host maintenance
1. When you're ready, click **Allocate**.

Once the host is ready, you can proceed to launch an <NT>EC2</NT> Mac instance.
