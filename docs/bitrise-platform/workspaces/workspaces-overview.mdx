A <NT><GlossTerm baseform="workspace">workspace</GlossTerm></NT> is an environment that allows you to manage your <NT>Bitrise</NT> <NT><GlossTerm baseform="project">project</GlossTerm></NT>s and the team members working on the projects. You can create multiple <NT>workspaces</NT>, and you can be invited to <NT>workspaces</NT> by other <NT>Bitrise</NT> users.

To be able to add projects and run builds, you either need to be part of a <NT>workspace</NT>, or you have to be an outside contributor on an project's team.

You also need a <NT>workspace</NT> to have [a paid subscription plan](https://www.bitrise.io/pricing) on <NT>Bitrise</NT>. Each of your <NT>workspaces</NT> can have a different subscription plan which determines how many credits your <NT>workspace</NT>'s projects can use.
