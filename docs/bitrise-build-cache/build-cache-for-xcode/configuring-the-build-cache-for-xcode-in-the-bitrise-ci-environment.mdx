You need the **<NT>Bitrise</NT> [Build Cache for Xcode](https://github.com/bitrise-steplib/bitrise-step-activate-build-cache-for-xcode)** <NT><GlossTerm baseform="step">Step</GlossTerm></NT> to activate the <NT>Build Cache</NT> for your <NT>Xcode</NT> project. After the <NT>Step</NT> executes, <NT>Xcode</NT> tasks will automatically read from the <NT>build cache</NT> and push new entries, too.

<Tabs>
  <TabItem value="workflow-editor" label="Workflow Editor">

  1. Make sure your build runs on a <NT><GlossTerm baseform="stack">stack</GlossTerm></NT> with <NT>Xcode</NT> 26 or higher version installed: [Setting the stack for your builds](/en/bitrise-ci/configure-builds/configuring-build-settings/setting-the-stack-for-your-builds).
  1. <Partial_OpeningTheWorkflowEditor />
  1. Add the [**Build Cache for Xcode**](https://github.com/bitrise-steplib/bitrise-step-activate-build-cache-for-xcode) <NT>Step</NT> to your <NT><GlossTerm baseform="workflow">Workflow</GlossTerm></NT>.

     The <NT>Step</NT> should be before any <NT>Step</NT> that executes <NT>Xcode</NT> tasks. For example, **fastlane** or [**Xcode Archive & Export for iOS**](https://github.com/bitrise-steplib/steps-xcode-archive).
  1. Optionally, you can disable pushing new cache entries: set the **<NT>Push</NT> new cache entries** input to **false**.

     In read-only mode, your build only reads from the cache but doesn't update it.
  1. Click **<NT>Save changes</NT>**.

  </TabItem>
  <TabItem value="yaml" label="Configuration YAML">

  1. Make sure your build runs on a stack with <NT>Xcode</NT> 26 or higher version installed: [Setting the stack for your builds](/en/bitrise-ci/configure-builds/configuring-build-settings/setting-the-stack-for-your-builds).
  1. Add the `activate-build-cache-for-xcode` <NT>Step</NT> to your <NT>Workflow</NT>.

     The <NT>Step</NT> should be before any <NT>Step</NT> that executes <NT>Xcode</NT> tasks. For example, `fastlane` or `xcode-archive`.

     ```yaml
     your-workflow:
       meta:
         bitrise.io:
           stack: osx-xcode-26.0.x-edge
       steps:
         - git-clone: {}
         - activate-build-cache-for-xcode: {}
     ```

  </TabItem>
</Tabs>
