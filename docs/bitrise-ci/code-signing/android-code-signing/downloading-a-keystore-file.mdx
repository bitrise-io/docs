import NT from '@site/src/components/NT';

A keystore file is required for <NT>Android</NT> <NT>code signing</NT>. You can define the location of the keystore file of an <NT>Android</NT> app in your `build.gradle` file: [Android code signing in Gradle](/en/bitrise-ci/code-signing/android-code-signing/android-code-signing-in-gradle). You can upload your keystore file to <NT>Bitrise</NT> and use the **<NT>File Downloader</NT>** <NT>Step</NT> to download the keystore file from <NT>Bitrise</NT> and put it in the location defined in the `build.gradle` file.

:::note[The <NT>Android Sign</NT> <NT>Step</NT>]

If you use the **<NT>Android Sign</NT>** <NT>Step</NT> to sign your app, you don't need to download the keystore file. The <NT>Step</NT> will find the file: [Android code signing using the Android Sign Step](/en/bitrise-ci/code-signing/android-code-signing/android-code-signing-using-the-android-sign-step).

:::

1. Add the **<NT>File Downloader</NT>** <NT>Step</NT> to your <NT>Workflow</NT>. The <NT>Step</NT> should be added BEFORE any <NT>Step</NT> that requires the keystore file, such as **<NT>Gradle Runner</NT>**.
1. Fill out the following two input fields:

   - **<NT>Download</NT> source url**: Set the generated <NT>keystore URL</NT> you get when you [upload your file to Bitrise](/en/bitrise-ci/code-signing/android-code-signing/uploading-android-keystore-files-to-bitrise)).
   - **<NT>Download</NT> destination path**: Set the location of the keystore file as a relative path. This path should be the same as the keystore path already defined in your `build.gradle` file (for example, `$HOME/keystores/project_release.keystore`).
1. Add the **<NT>Gradle Runner</NT>** <NT>Step</NT> right after your file downloading <NT>Step</NT>.

With that said, if you have successfully added the <NT>Steps</NT> to download your keystore file to the same location that you specified in your `build.gradle` file, you do not need the **<NT>Android Sign</NT>** <NT>Step</NT> in your workflow. Our **<NT>Gradle Runner</NT>** <NT>Step</NT> will sign and assemble your project.
