You can specify the <NT>code signing</NT> configuration for your project in [Android Studio](https://developer.android.com/studio/). You will need a keystore file, a <NT>key alias</NT> and a key password - have these ready before you start the procedure!

1. Open <NT>Android</NT> Studio.
1. Go to **Project navigator**.
1. Select your project and open **<NT>Module</NT> <NT>Settings</NT>**.
1. From **<NT>Module</NT>s**, select your module.
1. On the **Signing** tab, fill out the signing information. In our example, we used the following values:

   - Name: `release`
   - <NT>Key alias</NT>: `MyAndroidKey`
   - Key password: `***`
   - Store file: `/path/to/my/keystore.jks`
   - Store password: `***`

Once you filled out the signing information, the `signingConfigs` block will be created in your module’s `build.gradle` file. On <NT>Bitrise</NT>, you just need to build the app, either with the **<NT>Android Build</NT>** or the **<NT>Gradle Runner</NT>** <NT>Step</NT>.
