This guide describes how to manage your Android keystore files with the Bitrise API. If you’d like to learn more about how to do the same on the UI, please check out [Android code signing](urn:resource:component:42718).

| Endpoints | Function | Required role on the app's team |
| --- | --- | --- |
| GET/apps/\{app-slug\}/android-keystore-files | Get a list of Android keystore files | Owner or Admin |
| POST/apps/\{app-slug\}/android-keystore-files | Create an Android keystore file | Owner or Admin |
| DELETE/apps/\{app-slug\}/android-keystore-files/\{android-keystore-file-slug\} | Delete an Android keystore file | Owner or Admin |
