| Endpoints | Function | Required role on the app's team |
| --- | --- | --- |
| POST /apps/\{app-slug\}/builds | Trigger a new build. | Owner, Admin, or Developer |
| POST /apps/\{app-slug\}/builds/\{build-slug\}/abort | Abort a specific build. | Owner, Admin, or Developer |

You can <GlossTerm baseform="trigger">trigger</GlossTerm> and abort builds with the Bitrise API. Define parameters for the build: for example, branch, tag or git commit to use. Custom [Environment Variables](/en/bitrise-ci/configure-builds/environment-variables) can be defined as well.
