There are two main ways to handle dependencies for iOS apps:

- Using the [Swift Package Manager](https://developer.apple.com/documentation/xcode/swift-packages) which is built into Xcode.
- Using a third-party dependency manager, such as [Carthage](https://github.com/Carthage/Carthage) or [CocoaPods](https://cocoapods.org/).

Bitrise allows you to use either method.
