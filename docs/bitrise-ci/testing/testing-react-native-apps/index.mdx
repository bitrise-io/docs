- [Run unit and UI tests for React Native apps](/en/bitrise-ci/testing/testing-react-native-apps/running-unit-and-ui-tests-for-react-native-apps) by either `npm test` or `yarn test`. The <GlossTerm baseform="Step">Steps</GlossTerm> run the `tests` script in the `scripts` object of your package.
- [Perform end-to-end testing with Detox](/en/bitrise-ci/testing/testing-react-native-apps/running-detox-tests-on-bitrise.html), a gray box end-to-end tests and automation library for mobile apps built with React Native.

You can also use Bitrise's device testing solution, using Firebase Test Labs, to run device tests on both the iOS and Android apps of a React Native project: [Running device tests with Firebase for multiplatform apps](/en/bitrise-ci/testing/device-testing-with-firebase/running-device-tests-with-firebase-for-multiplatform-apps).
