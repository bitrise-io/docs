You can run all kinds of tests for your <NT>React Native</NT> app on <NT>Bitrise</NT>: unit tests, integration tests, or component tests. You can use [Jest](https://jestjs.io/), for example, to write all your tests and then run them during the build process with either [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/).

:::tip[E2E testing]

For end-to-end testing, check out Detox: [Running Detox tests on Bitrise](/en/bitrise-ci/testing/testing-react-native-apps/running-detox-tests-on-bitrise).

:::


<Tabs>
  <TabItem value="workflow-editor" label="Workflow Editor">

  1. Write your tests and add them to your <NT>React Native</NT> project.
  1. On <NT>Bitrise</NT>, add either the [**Run npm command**](https://github.com/bitrise-steplib/steps-npm) or the [**Run yarn command**](https://github.com/bitrise-community/steps-yarn) <NT><GlossTerm baseform="Step">Step</GlossTerm></NT> to your <NT><GlossTerm baseform="Workflow">Workflow</GlossTerm></NT>, depending on which package manager you use in your project.
  1. Configure the <NT>Step</NT> to run the `test` command:

     - For Yarn, find the **<NT>Arguments for running yarn commands</NT>** input and add `test`.

     - For npm, find the **<NT>The npm command with arguments to run</NT>** input and add `test`.

     ![npm-test.png](/img/_paligo/uuid-195577ac-fbba-5fe4-810b-3749e614682c.png)

     In either case, the <NT>Step</NT> will run the `test` script in the `scripts` object of your package.

  </TabItem>
  <TabItem value="yaml" label="Configuration YAML">

  1. Write your tests and add them to your <NT>React Native</NT> project.
  1. In the <NT><GlossTerm baseform="configuration yaml">Configuration YAML</GlossTerm></NT> file, add either the `npm` or the `yarn` <NT>Step</NT> to your <NT>Workflow</NT>, depending on which package manager you use in your project.
  1. Configure the <NT>Step</NT> of your choice to run the `test` command: for either <NT>Step</NT>, set the `command` input to `test`.

     ```yaml
     empty:
       steps:
         - git-clone: {}
         - npm:
             inputs:
             - command: test
         - yarn@0:
             inputs:
             - command: test
         - deploy-to-bitrise-io: {}
     ```

     Either <NT>Step</NT> will run the `test` script in the `scripts` object of your package.

  </TabItem>
</Tabs>


Run the tests and view their results: [Deploying and viewing test results](/en/bitrise-ci/testing/deploying-and-viewing-test-results).
