You can easily register a new Bitrise <GlossTerm baseform="Project">project</GlossTerm> from any command line interface: the process is guided and simple to follow. And it’s fast: if, for example, you have the `<GlossTerm baseform="bitrise.yml">bitrise.yml</GlossTerm>` file you want to use for the project, you do not have to wait for the <GlossTerm baseform="project scanner">project scanner</GlossTerm> to detect the project type and generate your `bitrise.yml` file. Just plug in the existing file and you are good to go!

:::tip[Adding a project with the API]

You can also use the Bitrise API to add a new project: [Adding and managing apps](/en/bitrise-ci/api/adding-and-managing-apps)

:::
