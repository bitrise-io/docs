<NT>Bitrise</NT> <NT><GlossTerm baseform="workflow editor">Workflow Editor</GlossTerm></NT> is designed in such a way that you can run it offline on your Mac/PC without having to log into [bitrise.io](https://www.bitrise.io/). The <NT><GlossTerm baseform="workflow">Workflow</GlossTerm></NT> Editor is open source, you can find its repository [here](https://github.com/bitrise-io/bitrise-workflow-editor). You can also join the discussion around <NT>Workflow Editor</NT> [here](https://discuss.bitrise.io/t/workflow-editor-v2-open-source-offline-workflow-editor/39).


## Installing the offline <NT>Workflow Editor</NT> to your computer

1. [Install the Bitrise CLI](/en/bitrise-ci/bitrise-cli/installing-and-updating-the-bitrise-cli).
1. Make sure you have [Go](https://golang.org/) installed on your local computer.
1. Run `bitrise setup` to install offline <NT>Workflow Editor</NT> as part of the <NT>Bitrise</NT> Plugins.

   Running `bitrise setup` also checks if <NT>Bitrise</NT> Core tools, <NT>Bitrise</NT> Plugins and Toolkits are installed on your local machine. If not, it will automatically install them. On macOS, it also checks for Homebrew, prompting you to install it manually if missing.


## Starting the offline <NT>Workflow Editor</NT>

1. `cd` into a directory where you have your `bitrise.yml`.
1. Run `bitrise :workflow-editor` command to start your offline session.


## Upgrading <NT>Workflow Editor</NT> version

You can upgrade to the <NT>latest version</NT> of the <NT>Workflow Editor</NT> by running `bitrise plugin update workflow-editor` command.
