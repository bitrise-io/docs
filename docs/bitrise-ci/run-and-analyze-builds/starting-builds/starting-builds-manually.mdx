## Rebuilding failed builds

Another way of starting a build manually is to rebuild a failed build. You can:

- Rebuild a Workflow build: [Rebuilding a failed Workflow build](/en/bitrise-ci/run-and-analyze-builds/build-data-and-troubleshooting/rebuilding-a-failed-build#rebuilding-a-failed-workflow-build).
- Rebuild a Pipeline, either the entire Pipeline or only unsuccessful Workflows: [Rebuilding a failed Pipeline](/en/bitrise-ci/run-and-analyze-builds/build-data-and-troubleshooting/rebuilding-a-failed-build#rebuilding-a-failed-pipeline).
- Rebuild with remote access: [Remote access](/en/bitrise-ci/run-and-analyze-builds/build-data-and-troubleshooting/remote-access).
