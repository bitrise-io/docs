If a build fails, you can rebuild any time to quickly recover from any errors. You can rebuild either single <GlossTerm baseform="Workflow">Workflows</GlossTerm> or an entire <GlossTerm baseform="Pipeline">Pipeline</GlossTerm>. A rebuild always uses the current <GlossTerm baseform="bitrise.yml">configuration YAML</GlossTerm> file: if a build fails and then you modify your configuration and rebuild, the rebuild will run with the new configuration.
