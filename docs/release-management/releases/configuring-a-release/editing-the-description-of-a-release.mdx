You can enter a release description when you add a new release. This helps identifying the release later. You can change the description at any time in the release configuration:

:::tip[Release description is internal only]

The release description is internal only, and it will not be included in the submission.

:::

1. <Partial_GettingToReleaseConfiguration />
1. On the release information card, click **Edit**.

   ![2025-08-08-rm-editing-release-description.png](/img/_paligo/uuid-54d3257f-0147-d101-0dde-c0296dcb798a.png)
1. In the **Edit Release details** dialog, update the description and click **Save**.
