Release Management

Bitrise Release Management simplifies distributing your iOS and Android apps to testers or directly to app stores—all from one platform, from the CI you choose.

[Overview](/en/release-management/getting-started-with-release-management/release-management-concepts)

[Quickstart](/en/release-management/getting-started-with-release-management)

Get started

To start using Release Management, you need to add at least one app.

To manage releases, connect your app to the App Store or Google Play.

You can use Release Management even if you don't use Bitrise CI.

Popular

Installable artifacts can be distributed for testing or used as release candidates.

Create release presets, set team permissions, and configure app store or LaunchDarkly integrations.

A Standard license grants access to features beyond the free plan's basic functionalities.

Featured

Distribute the builds of your mobile apps to testers without having to engage with either TestFlight or Google Play.

After connecting an app, you can add a new version and release it to the App Store or Google Play.

Bitrise provides a REST API for Release Management, which offers the same features as the GUI.
